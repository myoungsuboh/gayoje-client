## 티어별 월 마진 — **풀소진(base case)**, 3개 블렌드 민감도
가격: Pro $9 / Pro+ $19 / Pro Max $29. Paddle 차감 포함.

| 티어 | 토큰한도 | **캐싱 $0.52~0.65** | **중앙 $1.0** | **출력多 $1.5** |
|---|---|---|---|---|
| Pro $9 | 4M | **60~66%** | $4.05 (45%) | $2.05 (**23% ⚠️**) |
| Pro+ $19 | 8M | **65~70%** | $9.55 (50%) | $5.55 (29%) |
| Pro Max $29 | 12M | **66~72%** | $15.05 (52%) | $9.05 (31%) |

**판정 (더블체크 반영 — 캐싱은 입력만 깎고 출력비가 바닥을 깐다):**
- **캐싱 적용 + 입력 우세 → 풀소진이어도 60~72%. 건강.**
- 캐싱 없이 중앙 → 45~52%. AI 제품으론 생존 가능하나 빠듯.
- **출력 많음 → 23~31%(캐싱無)이고, 이땐 캐싱으로도 거의 못 살림** → 모델 라우팅 필요.

→ **풀소진이 base이면, 마진은 in:out 비율 + "비율에 맞는 레버(입력↑캐싱 / 출력↑라우팅)"가 결정한다.**

## 가장 중요 — 존망을 가르는 미지수
1. **파이프라인 1회전 실제 input:output 비율** — 23%냐 67%냐를 가름. **BE 실측 1순위.**
2. **컨텍스트 캐싱 적용 여부** — 우리 파이프라인은 CPS→PRD→설계로 직전 산출물을 재투입 → 캐싱 효과 최대. 풀소진 전제에선 **선택이 아니라 필수.**

## 비용 레버 (풀소진 마진을 안전선으로) — 우선순위
1. **🥇 컨텍스트 캐싱**(입력비 −90%): 단독으로 23~45% → 67~73%. **BE P0.**
2. **🥈 모델 라우팅**: 단순 단계(CPS 추출, 룰 생성)는 **Flash-Lite($0.05/$0.20, 10× 저렴)**, 복잡 합성(PRD·설계)만 Flash. 블렌드 추가 하락.
3. **🥉 quota 라이트사이징**: 캐싱으로도 부족하면 Pro 4M→~2M로 축소(풀소진 가정상 한도가 곧 원가). 단 offer 매력 ↓.
4. **가격 인상**: 4M 유지+65% 목표면 Pro는 ~$15여야. 전환율 리스크.
5. **연간 결제**: Paddle 고정 $0.50 분산 + 리텐션.

## 손익분기 (유닛 위)
고정 인프라(Vercel/Vultr/Neo4j ~$50–200/월)는 별도. 캐싱 적용 시 Pro 순익 ~$6 → **Pro 10~35명이면 고정비 커버.** 캐싱 없이 출력多($2)면 ~$50–200 / $2 = **25~100명** 필요 → 손익분기 난이도 급증.

## 초과팩 (풀소진 무관 — 단건)
Paddle 고정 $0.50 때문에 **$5 미만 팩 금지.** 권장: 미팅 +10건 $4.90, 처리량 +2M $5.90.

## 권장 결론 (rev2)
- **유닛 흑자는 가능. 단 "풀소진 + 캐싱 없음 + 출력 많음" 3중복이면 적자권(23%).**
- **그래서 캐싱/라우팅은 출시 전 필수 선결.** (BE.) 입력 우세면 캐싱으로 60~72%, 출력 우세면 라우팅(Flash-Lite)이 답 — in:out 측정이 선결.
- 가격은 Pro $9 / Pro+ $19 / Pro Max $29 유지 가능 — **캐싱 전제하에.** 캐싱 불가 시 가격 인상 or quota 축소 불가피.
- 출시 전 측정 2건: **(1) 실제 in:out 비율, (2) 캐싱 후 실블렌드 단가.**
